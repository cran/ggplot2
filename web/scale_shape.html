scale_shape

Discrete position scale

Details

See layer and qplot for more information on creating a complete plot from multiple components.

Examples


> dsmall <- diamonds[sample(nrow(diamonds), 100), ]
> 
> (d <- qplot(carat, price, data=diamonds, shape=cut))


> d + scale_shape(solid = TRUE)

 # the default
> d + scale_shape(solid = FALSE)


> d + scale_shape(name="Cut of diamond")


> d + scale_shape(name="Cut of\ndiamond")
