scale_area

Size scale for continuous variable

Details

See layer and qplot for more information on creating a complete plot from multiple components.

See also

  • position scale'>scale_discrete: Discrete position scales

Examples


> (p <- qplot(mpg, cyl, data=mtcars, size=cyl))


> p + scale for continuous variable'>scale_size("cylinders")


> p + scale for continuous variable'>scale_size("number\nof\ncylinders")


> 
> p + scale for continuous variable'>scale_size(to = c(0, 10))


> p + scale for continuous variable'>scale_size(to = c(1, 2))


> 
> # Map area, instead of width/radius
> # Perceptually, this is a little better
> p + scale_area()


> p + scale_area(to = c(1, 25))


> 
> # Also works with factors, but not a terribly good
> # idea, unless your factor is ordered, as in this example
> qplot(mpg, cyl, data=mtcars, size=factor(cyl))


> 
> # For lines, you need to tell that you want lines on the legend
> (p <- qplot(mpg, cyl, data=mtcars, size=cyl, geom="line"))


> p + scale for continuous variable'>scale_size(guide="line")
